# Sistema de Reserva para Restaurantes


1. Cadastro de Cliente; 
2. Login de Cliente; 
3. Criar Reserva; 
4. Cancelar Reserva; 
5. Pesquisar Restaurantes; 
6. Editar Reserva; 
	- Editar data ou horário; 
 	- Editar Mesa; 
 	- Editar número de pessoas; 
7. Gerenciar Reserva (Funcionário) 
 	- Confirmar a reserva; 
 	- Recusar a reserva; 
 	- Cancelar a reserva;
